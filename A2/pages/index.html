The following are the routes used in this assignment.

Routes Details
Method Route Description
GET / Displays the index page.
GET /api/todo Responds with an array listing of every ToDoEntry
GET /api/todo/open Responds with an array listing of every ToDoEntry with status of 'open'
GET /api/todo/completed Responds with an array listing of every ToDoEntry with status of 'completed'
GET /api/todo/:id Responds with a single ToDoEntry of particular 'id'
POST /api/todo Create and store a new ToDoEntry; if successful, it will respond with the new ToDoEntry in the response.
Request only accepts 'author', 'taskTitle' and 'taskDescription'
POST /api/todo/:id/notes Adds a note to the ToDoEntry with ':id' and responds with the updated entry.
Request should be provided as: {note: "The note text"}
PUT /api/todo/:id Updates the ToDoEntry with ':id' and responds with the updated entry.
Request only accepts 'taskTitle', 'taskDescription' and 'status'
DELETE /api/todo/:id Removes the ToDoEntry with ':id' and responds with the success/error response
